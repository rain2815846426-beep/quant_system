## 核心问题：为什么总收益是负的，但年化收益是正的？

### 问题 1：回测方法错误 ⚠️⚠️⚠️

**当前实现的问题**：

```python
# 当前代码逻辑
for month in months:
    # 使用月初价格选股
    selected = select_stocks(month_start)
    
    # 但是计算收益时
    portfolio_ret = (month_end_price - month_start_price) / month_start_price
```

**问题**：
1. **使用了未来数据** - 月初的价格在月初是未知的
2. **收益计算错误** - 应该是 t+1 日的收益，而不是当月收益
3. **调仓逻辑混乱** - 月度调仓但使用了当月数据

**正确的做法**：
```python
# t 日收盘后选股
selected = select_stocks(date=t)

# t+1 日开盘买入
entry_price = open_price[t+1]

# 持有到调仓日
exit_price = open_price[next_rebalance_date]
```
