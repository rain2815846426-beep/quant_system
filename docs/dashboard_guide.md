## 📊 功能模块

### 1. 📊 系统概览

**显示内容**:
- 股票数量、数据量统计
- 系统功能完成状态
- 快速操作按钮

**用途**: 快速了解系统状态

### 2. 🔄 数据更新

**功能**:
- ✅ 更新日线数据（获取最新行情）
- ✅ 计算因子（自动计算 57 个因子）
- ⚠️ 更新行业数据（可选，较慢）

**使用方法**:
1. 勾选需要更新的内容
2. 点击"🚀 开始更新"
3. 在终端查看实际进度

**实际更新命令**（在终端运行）:
```bash
# 更新日线数据
python3 scripts/update_daily.py

# 计算因子
python3 scripts/calculate_factors.py

# 更新行业数据
python3 scripts/download_real_industry.py
```

### 3. 📈 选股推荐

**功能**:
- 选择选股数量（10-50 只）
- 选择策略（反转/动量/多因子/机器学习）
- 行业中性化选项
- 导出 CSV

**使用方法**:
1. 设置选股数量（默认 20 只）
2. 选择策略（推荐"反转因子"）
3. 点击"🔍 生成选股推荐"
4. 查看推荐股票列表
5. 点击"📥 导出 CSV"下载

**输出示例**:
```
排名  股票代码    因子得分
1    000001.SZ   2.345
2    000002.SZ   2.123
3    600519.SH   1.987
...
```

### 4. 🤖 模型预测

**功能**:
- 选择模型（RandomForest/LightGBM/XGBoost）
- 选择预测周期（5/10/20 日）
- 查看模型评估指标

**使用方法**:
1. 选择模型（推荐 RandomForest）
2. 选择预测周期（推荐 20 日）
3. 点击"🔮 运行预测"
4. 查看 IC、RankIC、IC IR 等指标

**实际运行命令**（在终端运行）:
```bash

```

## 💡 日常使用流程

### 每日例行（5 分钟）

1. **打开 Dashboard**
   ```bash
   ./start_dashboard.sh
   ```

2. **更新数据**
   - 点击侧边栏"🔄 数据更新"
   - 勾选"更新日线数据"
   - 点击"🚀 开始更新"
   - （实际在终端运行更新脚本）

3. **获取选股推荐**
   - 点击"📈 选股推荐"
   - 设置选股数量（20 只）
   - 选择策略（反转因子）
   - 点击"🔍 生成选股推荐"
   - 导出 CSV

4. **查看模型预测**（可选）
   - 点击"🤖 模型预测"
   - 选择 RandomForest
   - 点击"🔮 运行预测"
   - 查看 IC 指标

### 每周分析（15 分钟）

1. **查看参数分析**
   - 点击"⚙️ 参数分析"
   - 查看特征重要性变化
   - 查看回测结果

2. **数据查询**
   - 查询关注股票
   - 查询因子走势
   - 查询行业分布

## 🐛 常见问题

### Q: Dashboard 打不开？

A: 检查 Streamlit 是否运行：
```bash
ps aux | grep streamlit
```

重启 Dashboard：
```bash
pkill -f streamlit
./start_dashboard.sh
```

### Q: 数据不更新？

A: 在终端手动运行更新脚本：
```bash
python3 scripts/update_daily.py
```

### Q: 选股推荐为空？

A: 先计算因子：
```bash
python3 scripts/calculate_factors.py
```

### Q: 图表不显示？

A: 检查 Plotly 是否安装：
```bash
pip3 install plotly
```
