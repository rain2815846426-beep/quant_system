# 蒙特卡洛参数敏感性分析报告

## 实现内容

### 1. 蒙特卡洛参数敏感性测试

**脚本**: `scripts/monte_carlo_analysis.py`

**功能**:
- 随机抽样 1000 组参数组合
- 自动运行回测
- 分析参数对结果的影响

**参数分布**:
- `top_n`: [10, 50] 均匀分布
- `stop_loss`: [10%, 25%] 均匀分布
- `max_position`: [5%, 15%] 均匀分布
- `target_vol`: [15%, 30%] 均匀分布

**输出**:
- `monte_carlo_results.csv` - 详细结果
- `monte_carlo_results.png` - 可视化图表

## 相反变量法（Antithetic Variates）

### 原理

对于每条随机路径 X，同时使用其镜像路径 -X：
- 原始路径：f(X)
- 镜像路径：f(-X)
- 平均：(f(X) + f(-X)) / 2

### 效果

- 方差降低 50-75%
- 相同精度下减少 50% 模拟次数

已在 `monte_carlo_backtest()` 中实现，通过随机种子控制可重复性。

## 结论

1. **避免过拟合** - 如果参数稍有变化策略就失效，说明过拟合

3. **量化不确定性** - 给出 95% 置信区间
