4. **数据量过大**: 全市场下载需要 45-60 分钟（已优化：只下载成分股）

### 优化决策

**从全市场下载 → 核心指数成分股**

理由：
1. 成分股代表性强（覆盖 A 股约 70% 市值）
2. 数据质量高（大盘股数据可靠）
3. 研究效率高（数据量小，计算快）
4. 适合个人研究（无需全市场数据）

效果：
- 下载时间：45-60 分钟 → 10-15 分钟
- 数据库体积：~13 GB → ~2-3 GB
- 股票数量：5,489 只 → ~1,300 只
