# 因子研究系统（Phase 2.5）

## 概述

因子研究系统用于评估因子的有效性，包括 IC 分析、RankIC 分析和因子分层回测。

## 使用方法

### 1. 分析所有因子

```bash
cd /Users/yuanxiaoyu/Downloads/quant_system

# 分析所有因子（IC + RankIC）
python3 scripts/analyze_factors.py
```

### 2. 分析单个因子

```bash
# 分析指定因子
python3 scripts/analyze_factors.py --factor momentum_20

# 指定预测周期
python3 scripts/analyze_factors.py --factor momentum_20 --forward_period 10
```

### 3. 指定日期范围

```bash
python3 scripts/analyze_factors.py --start_date 20200101
```

## 输出指标说明

### IC（Information Coefficient）

| 指标 | 说明 |
|------|------|
| **IC Mean** | IC 均值，衡量因子预测能力 |
| **IC Std** | IC 标准差，衡量稳定性 |
| **IC IR** | IC 信息比率 = IC Mean / IC Std |

### RankIC

| 指标 | 说明 |
|------|------|
| **RankIC Mean** | 秩相关均值 |
| **RankIC Std** | 秩相关标准差 |
| **RankIC IR** | 秩相关信息比率 |

### 分层回测

| 指标 | 说明 |
|------|------|
| **Q1-Q5** | 5 组收益（Q1 最低，Q5 最高） |
| **Long-Short** | 多空收益（Q5 - Q1） |
| **Cumulative** | 累计收益 |

## 因子有效性评估标准

### IC 评估

| IC IR | 评价 |
|-------|------|
| > 0.5 | 优秀 |
| 0.3 - 0.5 | 良好 |
| 0.1 - 0.3 | 可用 |
| < 0.1 | 无效 |

### RankIC 评估

| RankIC Mean | 评价 |
|-------------|------|
| > 0.1 | 优秀 |
| 0.05 - 0.1 | 良好 |
| 0.02 - 0.05 | 可用 |
| < 0.02 | 无效 |

## 当前分析结果解读

根据当前 7 个因子的 IC 分析结果：

1. **所有因子 IC 均为负值**
   - 说明因子值与未来收益呈负相关
   - 可能需要反向使用因子（低值买入）

2. **IC IR 普遍较低**
   - 最高：momentum_5 (-0.07)
   - 最低：volume_mean_20 (-0.63)
   - 说明因子预测能力不稳定

3. **可能原因**
   - 数据时间跨度太长（包含 2010-2018 年）
   - 市场风格变化
   - 因子需要优化或组合使用
