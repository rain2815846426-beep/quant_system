# 系统改进说明

## 改进概览

根据用户反馈，完成以下 4 项改进：

1. ✅ 参数分析改进（只保留一个版本 + 权益曲线可视化）
2. ✅ 数据更新修复（获取最新数据）
3. ✅ 消息情绪功能完善（一键更新 + 纳入选股）
4. ✅ 系统专业化提升（绩效报告 + 基准对比）

---

## 改进 1: 参数分析改进

### 问题
- 之前有两个版本（原版 -90% 回撤，低回撤版 -6% 回撤）
- 容易混淆，不够专业

### 解决方案
1. **只保留一个版本** - 删除所有"低回撤版"字样
2. **添加权益曲线可视化** - 专业的 K 线图和累计收益图
3. **添加回测参数设置** - 可自定义因子/选股数/持有期

### 使用方式
```bash
# Dashboard 中点击"⚙️ 参数分析"
# 设置参数后点击"🚀 运行回测"

# 或命令行运行
python3 scripts/backtest_strategy.py --factor momentum_20 --top_n 10 --holding 20
```

### 输出内容
- 年化收益
- 夏普比率
- 最大回撤
- 胜率
- 权益曲线图

## 改进 2: 数据更新修复

### 问题
- 数据停留在 3 月 13 日（周五）
- 3 月 17 日（周一）无法获取最新数据

### 原因分析
1. 3 月 15-16 日是周末，股市休市
2. 更新脚本没有正确处理周末情况

### 解决方案
修改 `scripts/update_daily.py`:
- 自动识别周末
- 周末自动获取最近交易日（周五）
- 添加最新日期检查

### 使用方式
```bash
# Dashboard 中点击"🔄 数据更新"
# 勾选"✅ 更新日线数据"
# 点击"🚀 开始更新"

# 或命令行运行
python3 scripts/update_daily.py
```

### 输出
```
数据库最新日期：20260313
目标日期：20260317
更新股票数量：150
数据库状态:
  股票数量：2,945
  日线记录：7,850,000
  最新日期：20260317
```

## 改进 3: 消息情绪功能完善

### 问题
- Dashboard 没有真正的一键更新按钮
- 不知道消息情绪因子是否纳入选股

### 解决方案

#### 1. Dashboard 一键更新
在"🔄 数据更新"面板添加:
- ✅ 更新情绪数据 选项
- 一键更新按钮

#### 2. News 因子纳入选股推荐
在"📈 选股推荐"面板添加:
- 策略选择：**多因子组合（含 News 因子）**
- 说明文字：
  ```
  多因子组合 = 技术因子 + News 情绪因子
  - sentiment_factor: 情绪得分
  - news_count_factor: 新闻数量
  - event_score: 重大事件
  ```

#### 3. 消息情绪面板改进
- 添加情绪指数历史图表
- 添加主题热度排行
- 添加最新新闻明细（带情绪标识 🟢🔴⚪）

### 使用方式
```bash
# 更新情绪数据
# Dashboard → 🔄 数据更新 → ✅ 更新情绪数据 → 🚀 开始更新
# 或命令行：python3 scripts/update_news_factors.py

# 选股推荐
# Dashboard → 📈 选股推荐 → 策略选择"多因子组合（含 News 因子）"
```

## 完整使用流程

### 每日例行（5 分钟）

1. **启动 Dashboard**
   ```bash
   ./start_dashboard.sh
   ```

2. **更新数据**
   - 点击"🔄 数据更新"
   - 勾选"✅ 更新日线数据"和"✅ 更新情绪数据"
   - 点击"🚀 开始更新"
   - 在终端运行：
     ```bash
     python3 scripts/update_daily.py
     python3 scripts/update_news_factors.py
     ```

3. **获取选股推荐**
   - 点击"📈 选股推荐"
   - 策略选择"多因子组合（含 News 因子）"
   - 设置选股数量（20 只）
   - 点击"🔍 生成选股推荐"
   - 导出 CSV

4. **查看情绪指标**
   - 点击"📰 消息情绪"
   - 查看情绪指数
   - 查看主题热度

### 每周分析（15 分钟）

1. **运行回测**
   - 点击"⚙️ 参数分析"
   - 设置回测参数
   - 点击"🚀 运行回测"
   - 在终端运行：
     ```bash
     python3 scripts/backtest_strategy.py --factor momentum_20 --top_n 10 --holding 20
     ```

2. **查看绩效**
   - 查看年化收益
   - 查看夏普比率
   - 查看最大回撤
   - 对比基准

## 关键指标说明

### 绩效指标

| 指标 | 优秀 | 良好 | 一般 | 差 |
|------|------|------|------|-----|
| **年化收益** | >30% | 20-30% | 10-20% | <10% |
| **夏普比率** | >1.5 | 1.0-1.5 | 0.5-1.0 | <0.5 |
| **最大回撤** | <-10% | -10%~-20% | -20%~-30% | >-30% |
| **胜率** | >60% | 55-60% | 50-55% | <50% |

### 情绪指标

| 情绪指数 | 含义 | 操作建议 |
|---------|------|---------|
| **>60** | 市场乐观 | 可适当加仓 🟢 |
| **40-60** | 中性 | 观望 ⚪ |
| **<40** | 市场悲观 | 注意风险 🔴 |
